## User Flows

### Casual User Flow (滑鼠操作)
```
1. 點擊「客戶管理」選單 → 2. 系統載入客戶列表（預設顯示全部）
3. 在「名稱」欄位輸入關鍵字 → 4. 點擊「🔍 查詢」
5. 點擊客戶列 → 6. 展開動作按鈕 → 7. 點擊「檢視」或「修改」
```

### Power User Flow (鍵盤快捷)
```
1. Ctrl+Shift+C        → 直接新增客戶（跳過列表）
2. Ctrl+F              → 焦點跳至統一編號搜尋框
3. ↓/↑                 → 在列表中上下移動
4. Enter               → 開啟目前列客戶
5. Ctrl+E              → 編輯目前列客戶（捷徑：顯示於工具提示）
```

## Taiwan Localization Details

| 項目 | 顯示格式 | 說明 |
|------|----------|------|
| 統一編號 | 8位數字 | 即時MOD11驗證，輸入第8碼後顯示✓或⚠ |
| 帳餘 | 千分位 + 負數紅字 | 負數顯示為紅色文字 |
| 狀態 | 正常/停用/待審核 | 三態標籤，Disabled灰色 |
| 列表標題 | 繁體中文 | 全域使用noto-sans-tc |

## shadcn/ui Components

- **Table**: `shadcn Table` with sortable columns (公司名稱, 帳餘)
- **Input**: 搜尋列使用 `shadcn Input` + `Form` 包裹
- **Select**: 狀態篩選使用 `shadcn Select` (全部/正常/停用)
- **Button**: 主要按鈕 `shadcn Button` (variant: default), 次要 `outline`
- **Badge**: 狀態顯示使用 `shadcn Badge` (green=正常, gray=停用)
- **Dialog**: 檢視/修改使用 `shadcn Dialog`
- **Tooltip**: 快捷鍵說明使用 `shadcn Tooltip`

## Tauri Integration Notes

- 列表分頁使用 local SQLite query，透過 Tauri command 橋接
- 統一編號驗證為純JS utility (MOD11)，不需Rust端處理
- 快捷鍵註冊：`tauri-plugin-global-shortcut`
- 建議：Table虛擬化使用 `react-virtual` 或 TanStack Table virtual，5000+列流暢捲動

## Keyboard Shortcuts (Tooltip)

| 快捷鍵 | 功能 | 說明 |
|--------|------|------|
| `Ctrl+Shift+C` | 新增客戶 | 任何畫面皆可 |
| `Ctrl+F` | 聚焦搜尋 | 統編或名稱欄位 |
| `↑/↓` | 上下導航 | 列表內移動 |
| `Enter` | 開啟詳情 | 進入檢視/編輯 |
| `Ctrl+E` | 編輯 | 快速編輯目前列 |
| `Esc` | 關閉彈窗 | 關閉Dialog/Sheet |

## Responsive Behavior

- 桌面優先 (>1280px): 完整欄位顯示
- 平板 (768-1280px): 隱藏電話欄位
- 手機 (<768px): 僅顯示統一編號+名稱+狀態，點擊展開其餘資訊
