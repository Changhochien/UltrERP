## User Flows

### Casual User Flow (滑鼠操作)
```
1. 點擊「庫存管理」選單 → 2. 點擊「庫存查詢」
3. 輸入產品代號或名稱 → 4. 點擊「🔍 查詢」
5. 系統顯示庫存資訊 → 6. 若低於安全存量，顯示警告
7. 點擊「補貨建議」→ 8. 查看建議補貨量
9. 可點擊「查看供應商」→ 10. 聯繫供應商補貨
```

### Power User Flow (鍵盤快捷)
```
1. Ctrl+Shift+L        → 開啟庫存查詢（全域）
2. Ctrl+F              → 聚焦產品代號欄位
3. Enter               → 執行查詢
4. Ctrl+P              → 列印庫存報表
5. ↓/↑                 → 瀏覽多項查詢結果
6. Ctrl+R              → 刷新目前庫存資料
```

## Taiwan Localization Details

| 項目 | 顯示格式 | 說明 |
|------|----------|------|
| 日期 | ROC格式 `YYY/MM/DD` | 庫存異動記錄 |
| 數量 | 整數 + 負數表示出庫 | +100 入庫, -10 出庫 |
| 產品代號 | alphanumeric | 系統自動產生 |
| 倉庫 | 下拉選單 | 總倉/分倉/在途 |

## shadcn/ui Components

- **Table**: 庫存異動記錄，使用虛擬化處理大量資料
- **Input**: 產品代號、產品名稱搜尋
- **Select**: 倉庫選擇 (總倉/分倉/在途)
- **Button**: `primary` (調整庫存), `secondary` (補貨建議), `outline` (庫存報表)
- **Alert**: 低於安全存量警告 (variant: warning)
- **Badge**: 庫存狀態 (正常=green, 過低=yellow, 緊急=red)
- **Card**: 庫存摘要資訊
- **Sheet**: 調整庫存/補貨建議側邊面板

## Tauri Integration Notes

- 庫存資料查詢：Tauri command 橋接 SQLite
- 低庫存 Alert：系統 Tray 可設定通知規則
- 建議：使用 TanStack Table + react-virtual 處理5000+ SKU
- 分頁：前端虛擬滾動，後端分頁 query

## Inventory Alert Logic

```
狀態判定:
- 正常:   在庫 > 安全存量
- 警告:   在庫 <= 安全存量 且 在庫 > 安全存量 * 0.5
- 緊急:   在庫 <= 安全存量 * 0.5
- 缺貨:   在庫 = 0

補貨建議公式:
建議補貨量 = 安全存量 * 2 - 目前在庫 + 預留
(最小訂購量: 50 條)
```

## Alert Display Options

| 頻率 | 建議UI模式 |
|------|------------|
| <5% SKU低於安全存量 | Alert Banner (單一產品) |
| 5-20% SKU低於安全存量 | 庫存儀表板 + 警示圖示 |
| >20% SKU低於安全存量 | 獨立「注意清單」檢視 + 批次動作 |
